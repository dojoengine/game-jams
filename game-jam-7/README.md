# Game Jam VII

**Date:** November 28-30, 2025

**Prize Pool:** $15,000

**Submissions:** 24 projects (highest turnout ever)

**Sponsors:** Starknet, StarkWare, Cartridge

**Winners Announcement:** https://x.com/ohayo_dojo/status/1995548529927094746

## Winners

### Co-Champions ($2,500 each)

**Kanoodle Fusion** by @dpinoness, @pilito_06
Commodore 64 vibes.
Spatial puzzle game where you place translucent polyomino pieces that stack and blend colors to match the target.
50 levels of solving shape and shade.

**Survivor Valhalla** by @0xMugen_, @ETazou
Extends Loot Survivor.
Composable auto-battler where you use ERC-721s from Loot Survivor to create a "beast base" and build a lineup to attack other players' bases.

### Second Place ($1,500 each)

**Starkcity** by @zarah_0x
Onchain Monopoly-style game where properties are ERC-1155 tokens.
Every dice roll, property trade, and rent payment is verifiably fair.

**Ghost Grid** by @Karan_Bisht09
Phasmophobia-inspired ghost-hunting game.
Hunt for evidence on a grid with all runs, evidence, and scores recorded onchain.
