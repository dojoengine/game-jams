### Project Summary
> Give a short summary of your game project.
Adventure is a gamified quest platform built on Starknet that combines onchain games, cryptographic puzzles, and dynamic NFTs. Players progress through sequential challenges including external Dojo-based games (NUMS, DopeWars, Loot Survivor) and cryptographic quest levels. The system creates an evolving treasure map NFT that visualizes player progression - as players complete levels, the on-chain NFT artwork dynamically updates, revealing new waypoints and connecting paths.

Key features:
- On-chain SVG generation in Cairo
- 64-bit bitmap progress tracking
- Signature-based puzzle verification
- Gasless transactions via Cartridge Controller
- Composable game integration via Denshokan interface
